# Zara AI 

Zara AI is a lightweight, high-performance chat interface., the application serves as a bridge to the Gemini(2.5 flash) model architecture via a API, providing users with an intuitive and responsive environment for AI-driven conversations.

# Used Technologies 

React: Used for building a dynamic, component-based user interface.

Vite: The frontend build tool providing an ultra-fast development experience.

Tailwind CSS: Provides a modern, utility-first styling framework for a sleek and responsive design.

Gemini AI: The underlying Large Language Model providing intelligent, high-speed reasoning.

Together AI API: The integration layer used to perform fast inference and manage model requests.
